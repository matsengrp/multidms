Installation¶

multidms requires Python 3.8 or higher.

The source code for multidms is available on GitHub at https://github.com/matsengrp/multidms.

The easiest way to install multidms is using pip:

… code-block:

pip install multidms

The fitting process can be quite computationally intensive, and if available, we recommend using a GPU to accelerate the fitting process. While multidms ships with jax and jaxlib as dependencies, these packages do not include CUDA support by default. For this, please update the jax installation in your environment to include CUDA support by following the instructions in the jax documentation.

pip install --upgrade "jax[cuda]" -f https://storage.googleapis.com/jax-releases/jax_cuda_releases.html

Developer install¶

git clone git@github.com:matsengrp/multidms.git
(cd multidms && pip install -e '.[dev]')
